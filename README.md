## 수업 구성 (8 Movements)

| # | Movement | 내용 |
|---|----------|------|
| 0 | **Prelude** | 오늘의 수업 커버 & 목표 |
| I | **큰 그림** | AI(기초수학) vs 웹빌더(응용수학) |
| II | **세 가지 길** | 아임웹 · 카페24 · 워드프레스 비교 (워드프레스 추천) |
| III | **프로젝트 정체** | 클래식 뉴스·인터뷰 사이트 브리프 + Gramophone/Bachtrack 벤치마크 |
| IV | **SEO 3분 이해** | 구글 노출 핵심 4가지 |
| V | **완성 미리보기** | 실제 사이트 목업 (CLASSICA) |
| VI | **용어 사전** | Domain, Hosting, WordPress, CMS, SEO, Theme |
| Finale | **로드맵** | 6단계 실행 계획 |

## 핵심 메시지

> "코드를 외우는 수업이 아닙니다.  
> 글을 올릴 줄 알면, 사이트는 이미 완성입니다."

- **도메인**: 카페24에서 이미 구매 완료 → 워드프레스에 연결
- **추천 스택**: 카페24 도메인 + WordPress CMS + SEO 플러그인
- **목표**: 영문 보도기사·인터뷰를 꾸준히 올려 구글 노출 최대화

## 기술 스택 (이 프레젠테이션)
- Pure HTML/CSS/JS (no framework)
- Playfair Display + Noto Serif/Sans KR
- 다크 네이비 + 골드 클래식 테마
- 키보드 좌우 화살표 / 버튼으로 슬라이드 이동
- 모바일 반응형 사이드바
